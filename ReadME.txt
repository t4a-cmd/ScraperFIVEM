🚀 Installation Guide

1️⃣ Step 1: Install the Required Modules

Run the setup script to install all necessary Python modules:

- Option 1: 🖱️ Double-click `setup.bat` to automatically install the dependencies.
- Option 2: 💻 Alternatively, you can manually run the following command in your terminal:

python install_modules.py


2️⃣ Step 2: Configure the Server IDs

📝 Add your server IDs to the `serveur.txt` file.  
(📂 29,000 server IDs are already included!)

3️⃣ Step 3: Start the Program

▶️ Once everything is set up, launch the program by running `start.bat`.
